# SIGP - Sistema Integrado de Gestão Presidiária

## 📌 Sobre o Projeto
O **SIGP** é um software desenvolvido como Projeto Integrador do curso de **Engenharia de Computação na FAMETRO (Manaus)**. O objetivo é substituir fluxos manuais por uma plataforma digital segura para o monitoramento de unidades prisionais, focando na gestão de pavilhões e detentos.

## 🛠️ Stack Técnica
- **Front-end:** HTML5, CSS3 e JavaScript Puro (Vanilla).
- **Back-end:** Node.js.
- **Banco de Dados:** PostgreSQL.
- **Metodologia:** Scrum (Gestão via Trello).
